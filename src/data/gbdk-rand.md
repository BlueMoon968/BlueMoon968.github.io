# GBDK Random Number Generator

An improved random number generator addon for GBDK (Game Boy Development Kit), providing better pseudo-random sequences for Game Boy games.

## Improvements
- Better distribution
- Faster execution
- Easy integration
- Well-documented

## Technical Details
This library addresses the limitations of the default GBDK RNG, implementing a more robust algorithm suitable for game development on the original Game Boy hardware.
