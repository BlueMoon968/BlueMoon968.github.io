# Pixel Platformer

A retro-style platformer inspired by classic 8-bit games, featuring tight controls and challenging level design.

## Key Features
- Classic platforming mechanics
- Multiple worlds and levels
- Secret areas and collectibles
- Speedrun-friendly design
- Retro pixel art aesthetic

## Development Highlights
Built with Unity, this project helped me master 2D physics and level design principles. Special attention was paid to player feel and responsiveness.
