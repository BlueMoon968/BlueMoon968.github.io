# Hide And Seek

A puzzle game where you need to find hidden objects in increasingly complex scenes. Built with pure JavaScript and HTML5 Canvas.

## Features
- Multiple challenging levels
- Pixel art aesthetics
- Smooth animations
- Progressive difficulty

## Development
This was my first complete game release, focusing on clean code and engaging puzzle mechanics.
